Hexo + Github page

Node JS,Hexo,gitHub

  1. Node js 安裝 → NodeJs
  2. 安裝 Git
  3. 注冊 GitHub 及新建一個 Repository (yourname.github.io)
    一定要是(「你的帳號名」.github.io)
  4. 安裝 HEXO

安裝 HEXO

生成 配置 ssh key

以記事本打開 c:\users\xxx\id_rsa.pub 並複製貼上到github https://github.com/settings/keys

設定個人資料

配置 _config.yml 文件

提交文件至 Github Page

到 yourname.github.io 看到你的網站就成功啦!!!!
